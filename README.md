# TOI — Documents & Philosophy

**TOI** is an educational card deck based on the Ogura Hyakunin Isshu (100 classical Japanese poems), designed around the philosophy that meaning emerges from context rather than being predetermined.

This repository hosts publicly shareable documentation, design notes, and philosophical materials about TOI.

## What is TOI?

TOI carries a triple meaning:
- **問い** (*toi*) — an inquiry
- **toy** — something to play with
- **The Original Inquiry**

Each card presents a question without a correct answer. Students navigate the same 100 cards, but their choices create personal context. The same poem, the same 100 questions, different goals for every participant.

## Design Philosophy

- **答えは文脈から生まれる** — Meaning emerges from context
- **同じ歌、同じ100問、全員違うゴール** — Same poems, same 100 questions, different goals for everyone
- No single correct answer. Each student’s selection history forms a unique personal signature.
